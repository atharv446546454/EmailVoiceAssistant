# EmailVoiceAssistant

## 📌 Overview

EmailVoiceAssistant is a full-stack web application that allows users to manage emails using voice commands. It leverages speech recognition and text-to-speech technologies to create a hands-free email experience, improving accessibility and productivity.

This project demonstrates the integration of frontend interfaces, backend services, and voice-based interaction systems.

## 🚀 Features

* 🎤 Voice-based email composition
* 📧 Send emails using voice commands
* 📥 Read received emails via text-to-speech
* 🧠 Speech recognition for command processing
* 🌐 Full-stack architecture
* ⚡ Responsive and interactive UI

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/atharv446546454/EmailVoiceAssistant.git
cd EmailVoiceAssistant
```

### 2. Install dependencies

```
npm install
```

### 3. Run backend server

```
cd server
npm install
npm start
```

### 4. Run frontend

```
npm run dev
```

---

## ▶️ Usage

1. Open the app in your browser
2. Allow microphone access
3. Use voice commands to:

   * Compose emails
   * Send emails
   * Read inbox messages

---

## 🔒 Environment Variables

Create a `.env` file in the `server` folder:

```
EMAIL_USER=your-email@example.com
EMAIL_PASS=your-email-password
```
